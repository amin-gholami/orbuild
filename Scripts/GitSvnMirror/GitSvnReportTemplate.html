OpenRISC GitSvnMirror

This is a collection of read-only Git mirrors of OpenRISC Subversion repositories, inspired by a similar collection at http://git.apache.org/ (whose source code is at https://svn.apache.org/repos/infra/infrastructure/trunk/projects/git/).

Note that these Git mirrors are missing all Subversion properties except for svn:executable. Some Subversion features like empty directories are also missing. Subversion tags and branches are not imported either.
